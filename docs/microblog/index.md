# Microblog

A low-fat feed for things that don't need a full blog post: a video worth watching,
an article worth flagging, a one-line signal from the democracy landscape. The
[Blog](/blog/) stays for substantial writing; this is for everything smaller.

**Subscribe:**

- RSS/Atom: [feed_microblog_rss_created.xml](https://designingopendemocracy.com/feed_microblog_rss_created.xml)
- JSON Feed: [feed_microblog_json_created.json](https://designingopendemocracy.com/feed_microblog_json_created.json)
- Mastodon / Bluesky: this site is bridged via [Bridgy Fed](https://fed.brid.gy/) — follow
  `@designingopendemocracy.com@web.brid.gy` from either network. Note that Bridgy Fed bridges
  the whole site, not just this section, so you'll also see main blog posts.
